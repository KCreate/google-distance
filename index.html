google-distance is a small Polymer-element written in Javascript that allows the user to input any address or location into the two textfields. It then sends a request to the google maps javascript api and displays the duration and distance inside the element. The developer needs to set the api-key as an attribute. More info the attributes tab. Element attributes

origin: The origin where the user starts

destination: The destination where the user will end up

api-key: Your API-Key from the Google Developer Console

duration: The duration it takes to get from the origin to the destination

distance: Distance between the origin and destination

last-response: The last API-response as a json-string

validResults: Boolean that is true when the response is valid. ZERO_RESULTS or NO_ROUTE_FOUND will set this to false.
